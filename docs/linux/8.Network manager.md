# Network manager 란?
Network manager는 daemon으로 동작하면서 네트워크의 설정을 관장한다. 연결 감지 및 해제, 재시도와 같은 연결부터 다른 애플리케이션이나 daemon에게 네트워크 정보를 제공 또는 관리를 위한 권한 부여 등을 수행한다.
